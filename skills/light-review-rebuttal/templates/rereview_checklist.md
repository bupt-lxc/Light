# Re-review checklist

Use after paper-writing/result-analysis/citation/figure/typesetting return real
artifacts. The commitment checker verifies structure; a human must still judge
scientific adequacy.

| issue_id | verbatim concern | author claim | action status | claimed locator/run | independently observed change/result | verdict |
|---|---|---|---|---|---|---|
|  |  |  | PLANNED / IN_PROGRESS / DONE / DECLINED / NOT_APPLICABLE |  |  | FULLY_ADDRESSED / PARTIALLY_ADDRESSED / NOT_ADDRESSED / MADE_WORSE / CANNOT_VERIFY |

Rules:

- Source concern must match the canonical registry.
- `PLANNED` and `IN_PROGRESS` cannot be `FULLY_ADDRESSED`.
- DONE without a reachable locator is `CANNOT_VERIFY`.
- Experiment/analysis DONE without run provenance is `CANNOT_VERIFY`.
- A clarification may be `NOT_APPLICABLE` for manuscript change and still be
  fully answered when the cited manuscript locator/evidence is real.
- A reasonable reviewer request is not closed merely because the author
  disagrees; record the evidence or editor-ruling path.
- No rejection-driving root cause is silently dropped.
