# Photocatalytic Dye Kinetics Study

> A medium-sized English journal-style E2E science demo for Light Skills. The data are synthetic and reproducible; the paper is a showcase artifact, not a real experimental claim.

This project demonstrates how Light Skills can carry a science-oriented project from a research question to data, analysis, figures, and an English manuscript.

## Topic

**Response-surface constrained kinetic analysis of visible-light dye photodegradation on nitrogen-modified TiO2.**

The simulated experiment follows a common environmental chemistry pattern: visible-light photocatalytic degradation of a dye surrogate, UV-vis absorbance time series, pseudo-first-order kinetic fitting, response-surface analysis, diagnostic checks, and radical-scavenger interpretation.

## Reproduce

```powershell
cd projects\photocatalytic-dye-kinetics-study
$env:PYTHONUTF8="1"
python scripts\run_all.py
```

Outputs:

- `data/synthetic_uvvis_timeseries.csv`
- `data/fitted_rate_constants.csv`
- `outputs/analysis_summary.json`
- `figures/figure1_kinetics.png`

- `figures/figure3_diagnostics.png`

- `paper/main.tex`

## Honesty boundary

- The data are synthetic and generated by `scripts/generate_data.py`.
- The paper demonstrates Light Skills' workflow, not a real photocatalyst discovery.
- Reference entries use real DOI-bearing background and software sources; verify fit, formatting, and citation style for a real submission.
- For a real submission, users must replace the synthetic generator with actual experimental data, instrument metadata, uncertainty analysis, and domain-specific peer review.
